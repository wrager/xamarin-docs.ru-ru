-   **Структура документа** &ndash; показывает дерево мини-приложения, составляющих макета. Можно щелкнуть элемент в дереве, чтобы оно выбранных в конструкторе. Кроме того щелкнув элемент в дереве загружает свойства элемента в окне «Свойства».

Панель инструментов предоставляет доступ на следующие компоненты:

-   **Селектор устройства** &ndash; определяет набор квалификаторов, связанный с конкретного устройства, такие как размер экрана, разрешение и доступности клавиатуры. Кроме того, можно также добавлять и удалять новых устройств.

-   **Выбор темы** &ndash; выбирает тема пользовательского интерфейса для макета.

-   **Выбор конфигурации** &ndash; выбирает конфигурацию устройства, такие как *Книжная* или *Альбомная*.

-   **Параметры классификатора ресурсов** &ndash; открывает диалоговое окно, представленных в раскрывающемся меню для выбора *язык*, *режиме пользовательского интерфейса*, *ночь режим*, и *Round экрана* параметры.

-   **Параметры панели действий** &ndash; настраивает параметры для макета панели действий.

-   **Редактор** &ndash; открывает *редактор*, что позволяет настраивать элементы выбранной темой.

-   **Существенные бланк** &ndash; включает или отключает *бланке материал*. Элемент раскрывающееся меню рядом бланке материал открывает диалоговое окно, которое позволяет настраивать в сетке.

Эти элементы управления упрощают процесс для просмотра определенных областей пользовательского интерфейса в конструкторе:

## <a name="property-pad-sections"></a>Панель свойств одновременно

Панель свойств состоит из нескольких разделов, которые группируют аналогичные свойства &ndash; делает проще найти интересующие свойства:

-   **Мини-приложение** &ndash; свойства Main мини-приложения, такие как `id`, `visibility`, `text`и т. д. Свойства для управления содержимым мини-приложения обычно размещаются здесь.

-   **Scroll** &ndash; прокрутки свойств.

-   **Поведение** &ndash; флаги, которые установлены, как ведет себя мини-приложения.

Структура выбранного мини-приложения (в данном случае `LinearLayout`) также выделяются в рабочей области конструирования. Это полезно для выбора Просмотр групп, которые не всегда легко выбрать в рабочей области конструирования.

Структура документа поддерживает копирование и вставка, или можно использовать перетаскивание и drop. Перетаскивание поддерживается в структуре документа в область конструктора, а также из рабочей области конструирования в структуру документа. Кроме того щелкнув правой кнопкой мыши элемент в структуре документа отображает контекстное меню для этого элемента (же контекстного меню, которое появляется при щелчке правой кнопкой мыши этот же мини-приложения в рабочей области конструктора).
